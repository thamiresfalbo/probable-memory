# Probable Memory

A public archive site made in Rails.

## Stack

- [Bulma](https://bulma.io) for Styling
- [SQLite](https://www.sqlite.org/index.html) as Database
- [Sorcery](https://github.com/Sorcery/sorcery) for authentication
